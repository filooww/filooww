### [auto-vpn](https://github.com/filooww/auto-vpn) — Bash automation for macOS
One-command VPN connect/disconnect for macOS. Wraps `networksetup` and keeps credentials in Keychain, so you don't type them every time.
*Stack: Bash, macOS, networksetup*

### [zoom_helper](https://github.com/filooww/zoom_helper) — auto-join Zoom meetings
Python utility that reads a schedule and launches the right Zoom meeting at the right time. Useful for back-to-back calls and lectures.
*Stack: Python, cron / launchd*

### [crm-free-agents](https://github.com/filooww/crm-free-agents) — zero-install CRM helper
Bookmarklet that parses a DayPilot Scheduler CRM directly from the DOM and shows which support agents are free in real time. No install, no extension.
*Stack: JavaScript, DOM scraping*

### [studentlab](https://github.com/filooww/studentlab) — PHP learning project
A small educational web project built while learning PHP. Demonstrates routing, forms and basic templating.
*Stack: PHP*
